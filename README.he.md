# FV Accessibility

תוסף נגישות ל‑WordPress התואם **לתקן ישראלי 5568** (WCAG 2.1 AA) ו**לתקנה 35** לתקנות שוויון זכויות לאנשים עם מוגבלות (התאמות נגישות לשירות), התשע"ג‑2013.

> **גילוי נאות.** פסיקות בית המשפט המחוזי בתל‑אביב (2022 ואילך) קבעו במפורש כי תפריט נגישות — בין אם כ‑overlay ובין אם בכל צורה אחרת — **אינו** ממלא לבדו את דרישות תקן 5568. גם האתר עצמו חייב להיות נגיש: HTML סמנטי, alt לתמונות, תבנית הניתנת לתפעול במקלדת, ניגודיות צבעים מספקת וכו'. התוסף הזה מבצע **שינויים אמיתיים ב‑DOM** (מחלקה על `<html>` + סגנונות + JS), והוא רכיב אחד בתוכנית עמידה בתקן — לא התוכנית כולה.

## פיצ'רים

תפריט בן 36 פעולות מאחורי כפתור צף הניתן להתאמה, בנוסף ל‑5 פרופילים בלחיצה אחת, בנוסף לעמוד הצהרת נגישות וערוץ דיווח שמתחייבים לפי החוק.

| קטגוריה | פעולות |
|---|---|
| **פרופילים** | עיוורים, כבדי ראייה, עיוורי צבעים, מוגבלי קוגניציה, נכי מוטוריקה |
| **תוכן** | גודל טקסט (4 שלבים, עד +200%), ריווח שורות / מילים / אותיות (3 שלבים), גובה שורה, גופן קריא, גופן לדיסלקסיה, יישור טקסט (סבב), הגדלת תצוגה (3 שלבים, עד 150%), הגדלת כפתורים, הדגשת כותרות, הדגשת קישורים, הדגשת פוקוס, תיאור לתמונות (alt ככיתוב), הגדלת תכנים בריחוף |
| **צבע וניגודיות** | סבב ניגודיות (בהירה / כהה), מונוכרום, ניגודיות הפוכה, סבב רוויה (גבוהה / נמוכה), בורר צבעים מותאם (רקע / טקסט / כותרות) |
| **מדיה ותנועה** | ביטול הנפשות, הסתרת תמונות, חסימת הבהובים (WCAG 2.3.1), השתק מדיה |
| **ניווט וסמן** | סבב סמן גדול (שחור / לבן), ניווט מקלדת (טבעת פוקוס + קישור "דלג לתוכן" + Alt+1..9 לכותרות), מדריך קריאה, מיקוד קריאה, תצוגת קריאה, מבנה העמוד, סיכום עמוד |
| **איפוס** | ניקוי כל ההתאמות |

בנוסף: עמוד הצהרת נגישות (נוצר אוטומטית בהפעלה, מנוהל באמצעות קיצור), טופס דיווח בתוך התפריט (שולח אימייל לרכז הנגישות שהוגדר), סורק תקינות מובנה, ועדכוני גרסה אוטומטיים מ‑GitHub.

## התקנה

1. העתיקו את התיקייה `fv-accessibility/` אל `wp-content/plugins/`.
2. הפעילו מתפריט "תוספים". התוסף ייצור אוטומטית עמוד `/accessibility/`.
3. עברו ל‑**הגדרות → נגישות**, ובלשונית **הצהרת נגישות** מלאו את שם רכז/ת הנגישות, אימייל, טלפון ותפקיד. פרטים אלה נדרשים בתקנה 35 — הם מופיעים בעמוד ההצהרה הציבורי ומשמשים כיעד לאימייל הדיווחים.

## הגדרה

הכל נמצא תחת **הגדרות → נגישות**, בשש לשוניות:

- **מיקום** — מיקום נפרד לכפתור במחשב/בנייד (צד, עיגון, היסט אופקי/אנכי, גודל), קיצור מקלדת.
- **עיצוב** — צבע רקע ואייקון של הכפתור.
- **יכולות** — הפעלה/השבתה לכל פריט בתפריט בנפרד, כדי לבחור אילו התאמות יוצגו ללקוח/ה.
- **הצהרת נגישות** — פרטי רכז/ת הנגישות, שם העסק, פטורים. השינויים מעודכנים בעמוד ההצהרה מיידית (ללא צורך בשמירת תוכן עמוד; העמוד מכיל את הקיצור `[fv_accessibility_statement]`).
- **מתקדם** — עמודים מוחרגים (לא יציגו את הכפתור), CSS מותאם, אישור לניקוי הגדרות בהסרת התוסף.
- **בדיקת תקינות** — סורק שעובר על דף הבית ועל 5 העמודים האחרונים ובודק: alt חסר, `<html lang>` חסר, h1 חסר, דילוגים בהיררכיית כותרות, שדות טופס ללא label, קישור "דלג לתוכן" חסר, וקישורים גנריים. בדיקה ראשונית; לכיסוי WCAG מלא הכולל בדיקת ניגודיות מחושבת מומלץ להתקין את התוסף **axe DevTools** בדפדפן.

## הערות ארכיטקטוניות

- **אין שום קריאה לרשת חיצונית.** אין מעקב, אין AI overlay, אין סקריפט צד שלישי, אין תלות ב‑CDN.
- **הגדרות**: option בודד `fv_accessibility_settings`, מתמזג עומקית עם ברירות מחדל כך שהתקנות ישנות מקבלות מפתחות חדשים אוטומטית.
- **שמירה**: בחירות המבקר נשמרות ב‑`localStorage` עם נפילה ל‑cookie (`fv_a11y_state`, שנה, `SameSite=Lax`). הבחירות נשמרות בין דפים ובין ביקורים.
- **ללא הבזק תוכן לא מותאם**: סקריפט פנימי (~1KB) ב‑`<head>` קורא את ה‑cookie/localStorage ומחיל את מחלקות ה‑`fv-*` על `<html>` *לפני* הציור.
- **בידוד מגנן**: כללי ה‑CSS משתמשים בסלקטורים עם `:not(.fv-a11y-button) :not(.fv-a11y-panel)` כדי שההתאמות לא יזלגו לתפריט עצמו.
- **מלכודת ה‑containing block**: הוספת `filter` על `<body>` הופכת את body ל‑containing block של רכיבי `position: fixed`, מה שהיה תולש את התפריט מהויופורט. כללים של filter (מונוכרום, היפוך, רוויה) פועלים על `body > *:not(.fv-a11y-button):not(.fv-a11y-panel)` במקום.
- **התפריט אינו modal**: `aria-modal="false"` כדי שמקריאי מסך יוכלו להמשיך לנווט בעמוד שמתחת. Esc סוגר; הפוקוס חוזר לכפתור.
- **תנועה מופחתת**: כל המעברים וההגדלות בריחוף מאופסים תחת `@media (prefers-reduced-motion: reduce)`.
